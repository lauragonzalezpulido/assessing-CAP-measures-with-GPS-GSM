# assessing-CAP-measures-with-GPS-GSM
Code for assessing the effectiveness of conservation measures using GPS-GSM data. Example assessing the effectiveness of CAP measures using data from GPS-GSM tracked birds.
This code was developed for a scientific publication authored by Laura González-Pulido, David González del Portillo, Lara Moreno-Zarate, Gerard Bota, Gemma Clemente-Orta, Eladio García de la Morena, David Giralt, Carlos A. Martín, Manuel B. Morales, François Mougeot, João P. Silva, and Mario Díaz. Code authors: Laura González-Pulido, David González del Portillo, and Lara Moreno-Zarate.
